# Simple Web MP3 Player

### This simple web mp3 player can host it by yourself.  Just put two files in a folder and open explorer to run.

### A sample free audio book from LibriVox is for testing, mp3 url is parsed into mp3_urls.csv file with Collection, Title, MP3_URL. columns.

### It can run continously, shuffle or select yourself then continue, speed and download also available.

### You can play any mp3 by uploading csv file as required format, then go.

### Have a greate fun~

#### Sample page: https://mygithub.k2u.io 

#### https URL works fine on both mobile explorer and desktop, but http URL may not work on mobile.
